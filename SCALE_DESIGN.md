# ESP32 智能体脂秤设计方案
生成于: 2026-06-26

============================================================
一、系统架构
============================================================

ESP32(传感器) --WiFi--> /api/scale/* (server.js --SQLite--> DB)
                                |
                                | JSON API
                           index.html (体重趋势页)

============================================================
二、硬件方案（约185元）
============================================================

元件清单:
  - ESP32 Development Board
  - HX711 + 压力传感器 x4
  - AD5933 BIA 体脂测量模块
  - OLED 128x64 (I2C)
  - 18650 锂电池 + 充电模块

引脚:
  HX711 -> GPIO 19/18
  AD5933/OLED -> GPIO 21/22 (I2C)
  按键 -> GPIO 26/27

============================================================
三、数据模型
============================================================

weight_records 表:
  weight(kg), body_fat(%), muscle_mass(kg),
  water_pct(%), bone_mass(kg), bmr(kcal),
  bmi, impedance(ohm), device_id, recorded_at

scale_devices 表:
  device_id, user_id, calibration_factor, firmware

============================================================
四、API
============================================================

POST /api/scale/record  - 上传测量数据
GET /api/scale/history  - 获取历史 (Query: days)
GET /api/scale/latest   - 获取最新记录
POST /api/scale/register - 注册设备

============================================================
五、ESP32 固件流程
============================================================

1. 深度睡眠待机
2. 重量>5kg唤醒
3. 读取传感器 -> 总重量
4. 等待稳定
5. AD5933 BIA 测量
6. 计算体脂率
7. 连WiFi POST到服务器
8. OLED显示 -> 深度睡眠

============================================================
六、前端体重趋势页
============================================================

新页面: screen-weight (tab-bar第5个tab)
  - Chart.js: 30天体重+体脂折线图
  - 数据卡片: 最新体重/体脂/肌肉量/BMR
  - 7天/30天变化对比
  - 手动输入入口
