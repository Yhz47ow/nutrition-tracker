# Changelog

本项目使用语义化版本：修复递增补丁号，兼容功能递增次版本号，不兼容变更递增主版本号。

## [1.2.0] - 2026-07-13

### Added

- 内置六大肌群力量训练动作库与自定义动作管理。
- 多动作训练、重量/次数/完成状态记录及每动作独立组间休息。
- 使用结束时间戳校准的环形休息倒计时，支持暂停、跳过和增加 10 秒。
- Web Notification、Service Worker 提醒、提示音与震动反馈。
- 训练日历、训练详情和动作近期重量/次数趋势。
- 首页最近训练、时长和估算消耗概览。
- 训练数据本地迁移、文件/Gist 同步与可选 SQLite 后端同步。

### Compatibility

- 保留 `v1.1.0` 的全部饮食数据键和数据结构。
- 训练数据使用独立、版本化的 `localStorage` 键。

## [1.1.0]

- PWA 同步与离线缓存升级。
