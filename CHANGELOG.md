# Changelog

本项目使用语义化版本：修复递增补丁号，兼容功能递增次版本号，不兼容变更递增主版本号。

## [2.1.0] - Unreleased

### Added

- 新增原生微信小程序工程，包含首页、搜索、食物库、训练、记录和设置等 10 个页面。
- 迁移 71 种内置食物、联网食物查询、条码扫描、食物照片、自定义食物、营养目标和智能推荐。
- 迁移六大肌群动作库、自定义动作、多动作多组训练、时间戳休息倒计时、历史日历和动作趋势。
- 新增微信本地 JSON 备份导出、聊天文件导入，以及旧 PWA 备份字段兼容。
- 新增小程序页面/资源/事件校验和本地存储、饮食、训练自动测试。

### Changed

- 小程序数据统一使用 `wx.getStorageSync`、`wx.setStorageSync` 和用户文件目录，不依赖远端账号或令牌。

### Removed

- 小程序端不包含 GitHub/Gist、音乐、DOM、Service Worker 或 Web Notification 代码。

### Platform limits

- 微信被系统挂起后不能执行秒级后台任务；休息计时以结束时间戳校准，回到前台后准确恢复，但纯本地模式不能保证后台系统通知。

## [2.0.0] - Unreleased

### Added

- 新增 Capacitor 8 iOS/Android 原生工程，Web 资源随安装包内置并支持离线启动。
- Android 新增组间休息前台服务、精确闹钟、高优先级通知、声音和震动。
- iOS 新增基于 `UNUserNotificationCenter` 的系统级组间休息通知。
- 新增原生状态栏、安全区、启动页、Android 返回键和触觉反馈适配。
- 新增统一 App 图标、启动画面、Android release keystore 配置和原生发布文档。

### Compatibility

- 保留 PWA/Web 运行方式和现有饮食、训练数据结构。
- 原生 WebView 与浏览器存储域隔离，首次安装通过 JSON 或 GitHub Gist 迁移数据。

### Removed

- 删除训练音乐控制条、外部平台嵌入、本地播放器、白噪音和 iOS MusicKit 集成。

### Verification

- 当前开发机缺少完整 Xcode、Android SDK 和 JDK；正式 `v2.0.0` 发布前必须完成两端编译、签名和真机验收。

## [1.3.0] - 2026-07-13

### Added

- 训练界面新增始终可见的音乐控制条和音乐设置入口。
- Apple Music、网易云音乐官方嵌入播放器，以及 QQ音乐、小宇宙实验性 Web 嵌入与外部打开降级。
- 本地 URL 播放列表和本机音频文件支持上一首、播放/暂停、下一首及独立音量。
- 可离线使用的白、粉红、棕色噪音生成器。
- 本地媒体接入 Media Session，支持系统媒体面板和耳机按键。
- 组间休息最后三秒混音提示，不中断训练音乐。
- 音乐配置加入本地导出和 GitHub Gist 同步。

## [1.2.1] - 2026-07-13

### Added

- 训练中的动作仓库支持按胸部、背部、腿部、肩部、手臂和核心筛选。
- 全部动作视图按肌群分组展示，并保留跨分类多选。
- 自定义动作明确要求选择肌群分类，保存后进入对应分类。

## [1.2.0] - 2026-07-13

### Added

- 内置六大肌群力量训练动作库与自定义动作管理。
- 多动作训练、重量/次数/完成状态记录及每动作独立组间休息。
- 使用结束时间戳校准的环形休息倒计时，支持暂停、跳过和增加 10 秒。
- Web Notification、Service Worker 提醒、提示音与震动反馈。
- 训练日历、训练详情和动作近期重量/次数趋势。
- 首页最近训练、时长和估算消耗概览。
- 训练数据本地迁移、文件/Gist 同步与可选 SQLite 后端同步。

### Compatibility

- 保留 `v1.1.0` 的全部饮食数据键和数据结构。
- 训练数据使用独立、版本化的 `localStorage` 键。

## [1.1.0]

- PWA 同步与离线缓存升级。
